# ASTRYX

Multipurpose Discord bot with slash, prefix, and hybrid commands.

## Setup

1. Copy `.env.example` to `.env`.
2. Fill in the Discord bot token, client ID, owner ID, database URL, and any optional API keys.
3. Install dependencies with `npm install`.
4. Start the bot with `npm start`.

## Scripts

- `npm start` - run the bot.
- `npm run check` - syntax-check project JavaScript files.
